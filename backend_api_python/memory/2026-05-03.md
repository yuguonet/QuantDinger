### 单只K线完整调用链

```
KlineService.get_kline(market, symbol, timeframe, limit, adj)
  │
  ├─ 1. 查缓存 (cache.py)
  │     key = make_key("kline", symbol, timeframe, limit)
  │     cached = self._cache.get(key, data_type)
  │     ├─ 命中 → adjust_kline(symbol, cached, adj) → 返回
  │     └─ 未命中 ↓
  │
  ├─ 2. DataSourceFactory.fetch_kline_raw(symbol, tf, limit, market)
  │     │
  │     ├─ InflightDedup.get_or_submit(key, fn)
  │     │   ├─ 已有相同请求在跑 → 等结果，不重复发
  │     │   └─ 没有 → 提交新任务 ↓
  │     │
  │     ├─ _do_sequential_fallback(symbol, tf, limit, market)
  │     │   │
  │     │   ├─ get_providers("kline", timeframe=tf, market=market)
  │     │   │   → 按 priority 排序的 Provider 列表
  │     │   │   → 过滤: capability=kline + timeframe 匹配 + market 匹配
  │     │   │
  │     │   └─ sequential_fallback(symbol, providers, cb)
  │     │       → 按优先级逐源尝试:
  │     │         Tencent(10) → Sina(20) → EastMoney(30) → AkShare(50)
  │     │       → 熔断器跳过已熔断的源
  │     │       → 返回第一个成功的结果 + 源名
  │     │
  │     └─ 返回原始(未复权) K线数据
  │
  ├─ 3. 写缓存
  │     self._cache.set(key, raw, ttl, data_type)
  │     TTL 按周期: 1m→60s, 5m→120s, 1D→14400s, 1W→86400s
  │
  └─ 4. adjust_kline(symbol, raw, adj) → 复权后返回
```

KlineService.get_kline_batch(market, symbols, timeframe, limit)
  │
  ├─ 1. 逐只查缓存 → 分离 hit/miss
  │
  ├─ 2. DataSourceFactory.fetch_kline_batch(need_fetch, tf, limit, market)
  │     │
  │     ├─ get_providers("kline", ...) → Provider 列表

  │         │
  │         ├─ _get_available_sources() → 按 market 过滤可用源
  │         │
  │         ├─ _WorkQueue(symbols) → 动态阻塞队列
  │         │
  │         ├─ 多线程 worker 并发:
  │         │   每个 source 一个 worker 线程
  │         │   从队列取 symbol → fetch_kline → 成功/失败
  │         │   ├─ 成功: 记录结果，取下一个

### Provider 层详细

#### 自注册机制
```python
# provider/__init__.py
_registry: Dict[str, BaseDataSource] = {}  # 全局注册表

@register(priority=10)
class TencentDataSource:
    name = "tencent"
    capabilities = {
        "kline": True,
        "kline_tf": {"1m","5m","15m","30m","1H","1D","1W"},
        "quote": True,
        "batch_quote": True,
        "hk": True,
        "markets": {"CNStock", "HKStock"},
    }
```

- `@register` 装饰器: import 时自动实例化并注册到全局表
- `autodiscover()`: 扫描 provider/ 目录，import 所有模块触发注册
- `get_providers(capability, timeframe, market)`: 按条件过滤 + priority 排序

#### 各 Provider 能力

| Provider | priority | K线 | 行情 | 批量行情 | 港股 | 市场 |
|----------|----------|-----|------|----------|------|------|
| Tencent | 10 | 全周期 | ✅ | ✅(500/批) | ✅ | CN/HK |
| Sina | 20 | 日/周 | ✅ | ✅(500/批) | ❌ | CN |
| EastMoney | 30 | 日/周 | ✅ | ✅(6000/批) | ❌ | CN |
| AkShare | 50 | 日/周/分钟 | ❌ | ✅(全市场) | ❌ | CN |
| HKStock | 40 | 日 | ✅ | ❌ | ✅ | HK |

### 缓存层 (cache.py)

**两层存储**:
- **热数据** (行情/分钟线): 内存 OrderedDict (LRU)，丢了 30s 重取
- **温数据** (K线/股票信息): feather 文件，进程重启还在

**TTL 策略**:
| 数据类型 | TTL | 存储层 |
|---------|-----|--------|
| ticker/quote | 30s | 内存 |
| kline:1m | 60s | feather |
| kline:5m | 120s | feather |
| kline:1D | 4h | feather |
| kline:1W | 24h | feather |
| stock_info | 24h | feather |
| zt_pool/dt_pool | 60s | 内存 |
| dragon_tiger | 300s | 内存 |

### 复权层 (adjustment.py)

- Provider 只返回原始(未复权)数据
- 复权在 KlineService → adjust_kline 统一处理
- 除权除息数据从东财获取，缓存 24h
- 支持前复权(qfq) / 后复权(hfq) / 不复权

### 熔断器 (circuit_breaker.py)

- 每个数据源独立熔断状态
- 连续失败达到阈值 → 熔断一段时间
- 熔断期间跳过该源，直接尝试下一源
- 熔断恢复后重新尝试

### 限流器 (rate_limiter.py)

- 每个数据源独立限流（腾讯/东财/AkShare 各有不同频率）
- `get_tencent_limiter()` / `get_eastmoney_limiter()` / `get_akshare_limiter()`
- 超限时自动等待，避免被封 IP

## KlineService 架构与文件关联深度分析

### KlineService 定位

KlineService 是整个数据层的**缓存分水岭**：
- 上层（routes/services/agent）只跟 KlineService 交互
- 下层（data_sources/provider）被 KlineService 屏蔽
- 所有数据获取逻辑委托给 DataSourceFactory，自身只管缓存

### KlineService 公开接口

|------|------|---------|
| `get_kline(market, symbol, timeframe, limit, adj)` | 获取K线（单只/批量） | 最高 |
| `get_ticker(market, symbol)` | 获取实时行情（单只/批量） | 高 |
| `get_latest_price(market, symbol)` | 获取最新价格（简化接口） | 中 |
| `get_realtime_price(market, symbol, ...)` | 获取实时价格+涨跌信息 | 高 |
| `get_cache_dir()` | 获取缓存目录路径 | 低 |
| `invalidate(symbol, data_type)` | 清除缓存 | 低 |
| `cache_stats()` | 缓存统计 | 低 |
| `source_stats()` | 数据源统计 | 低 |
| `prewarm_all(market, symbols, ...)` | 预热缓存 | 低 |

### 调用链详解

#### 链路1: 单只K线 get_kline (最核心路径)

```
KlineService.get_kline(market, symbol, timeframe, limit, adj)
│
├─ [1] 查缓存
│    cache.get(make_key("kline", symbol, tf, limit), "kline:{tf}")
│    ├─ 命中 → adjust_kline(symbol, cached, adj) → return
│    └─ 未命中 ↓
│
├─ [2] DataSourceFactory.fetch_kline_raw(symbol, tf, limit, market)
│    │
│    ├─ InflightDedup.get_or_submit(key, fn)
│    │   └─ 防止同一 symbol 并发重复请求
│    │
│    └─ _do_sequential_fallback(symbol, tf, limit, market)
│        │
│        ├─ get_providers("kline", timeframe=tf, market=market)
│        │   → 按 priority 排序: Tencent(10) → Sina(20) → EastMoney(30) → AkShare(50)
│        │
│        └─ sequential_fallback(symbol, providers, cb)
│            ├─ 熔断器跳过已熔断的源
│            └─ 逐源尝试: provider.fetch_kline(symbol, tf, limit)
│                ├─ TencentDataSource.fetch_kline → qt.gtimg.cn / web.ifzq.gtimg.cn
│                ├─ SinaDataSource.fetch_kline → hq.sinajs.cn / quotes.sina.cn
│                ├─ EastMoneyDataSource.fetch_kline → push2his.eastmoney.com
│                └─ AkShareDataSource.fetch_kline → ak.stock_zh_a_hist / ak.stock_zh_a_hist_min_em
│
├─ [3] 写缓存
│    cache.set(key, raw, get_ttl("kline", tf), "kline:{tf}")
│    TTL: 1m→60s, 5m→120s, 15m→180s, 1H→300s, 1D→14400s, 1W→86400s
│
└─ [4] adjust_kline(symbol, raw, adj)
     ├─ adj="qfq" → 前复权
     ├─ adj="hfq" → 后复权
     └─ adj=""    → 不复权
     除权除息数据: 东财API, 缓存24h
```

#### 链路2: 批量K线 get_kline_batch

```
KlineService.get_kline_batch(market, symbols, timeframe, limit)
│
├─ [1] 逐只查缓存 → 分离 cached / need_fetch
│
├─ [2] DataSourceFactory.fetch_kline_batch(need_fetch, tf, limit, market)
│    │
│    ├─ get_providers("kline", ...) → Provider 列表
│    │
│    └─ Coordinator.coordinate_kline(symbols, tf, limit, providers, cb)
│        │
│        ├─ _get_available_sources(market, provider_map, cb)
│        │   → 按 market 过滤 + 熔断器检查
│        │
│        ├─ _WorkQueue(symbols) → 动态阻塞队列
│        │
│        ├─ 多线程 worker 并发 (每个 source 一个线程):
│        │   worker 线程:
│        │     loop:
│        │       sym = wq.get()         # 从队列取 symbol
│        │       bars = provider.fetch_kline(sym, tf, limit)
│        │       if 成功:
│        │         results[sym] = bars   # 记录结果
│        │         wq.task_done()       # 标记完成
│        │       if 失败:
│        │         if 还有其他源没试:
│        │           wq.put_back(sym)   # 放回队尾
│        │         else:
│        │           failed.append(sym) # 彻底失败
│        │
│        ├─ 吞吐反馈: 连续失败≥5次 → 暂停该源 30s
│        │
│        └─ return (results, failed)
│
├─ [3] 回调 on_raw_data(sym, raw_bars) → KlineService 写缓存
│
└─ [4] 逐只 adjust_kline(sym, bars, "qfq") → 合并返回
```

### 数据流转全过程

```
[用户请求]
    │
    ▼
[routes/kline.py]  HTTP GET /api/kline?symbol=SH600519&timeframe=1D&limit=100
    │
    ▼
[KlineService.get_kline("CNStock", "SH600519", "1D", 100, "qfq")]
    │
    ├─ cache.get("kline:SH600519:1D:100") → miss
    │
    ├─ DataSourceFactory.fetch_kline_raw("SH600519", "1D", 100, "CNStock")
    │    │
    │    ├─ InflightDedup: 无重复 → 提交
    │    │
    │    └─ sequential_fallback:
    │         get_providers("kline", "1D", "CNStock")
    │         → [Tencent(10), Sina(20), EastMoney(30), AkShare(50)]
    │
    │    └─ 尝试 TencentDataSource.fetch_kline("SH600519", "1D", 100)
    │         │
    │         ├─ rate_limiter.wait()  # 限流
    │         ├─ requests.get("https://web.ifzq.gtimg.cn/appstock/app/fqkline/get?...")
    │         ├─ 解析 JSON → [{time, open, high, low, close, volume}, ...]
    │         └─ return 100 bars
    │
    ├─ cache.set(key, raw_bars, ttl=14400, "kline:1D")  # 写缓存(4h)
    │
    └─ adjust_kline("SH600519", raw_bars, "qfq")  # 前复权
         │
         ├─ 获取除权除息数据 (东财API, 缓存24h)
         ├─ 计算复权因子
         ├─ 调整 OHLCV 价格
         └─ return 复权后 100 bars
    │
    ▼
[routes/kline.py]  返回 JSON 给前端
```
