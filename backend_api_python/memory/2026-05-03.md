## 项目架构概览

```
routes/ (API 路由层)
  ├── kline.py, market.py, portfolio.py, strategy.py ...
  └── 引用 services/ + data_sources/

services/ (业务逻辑层)
  ├── kline.py (KlineService - 缓存层，分水岭)
  ├── market_data_collector.py
  ├── symbol_name.py, strategy.py ...
  └── 引用 data_sources/

data_sources/ (数据源层)
  ├── factory.py (DataSourceFactory)
  ├── cache.py, normalizer.py, coordinator.py ...
  ├── cn_stock.py ← 关键文件，依赖旧 provider
  └── provider/ (新重构的类实现)
      ├── tencent.py (TencentDataSource)
      ├── sina.py (SinaDataSource)
      ├── eastmoney.py (EastMoneyDataSource)
      └── akshare.py, hk_stock.py
```
