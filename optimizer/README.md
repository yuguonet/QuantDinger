## 策略方向

### 优先级排序
1. **量价策略** — 量价关系是最显著特征，做"放量突破/缩量回调"
2. **多因子选股** — 全市场横截面分化大，波动/量价/趋势因子组合
3. **趋势跟踪** — 自相关中等偏强，中期趋势(5-20天)
4. **短线/日内** — 振幅3.54%够用，需15分钟数据配合

### 策略设计方向
- **量价共振策略**: 放量突破 + 趋势确认 + 波动率过滤
- **龙回头策略**: 大涨后回调缩量买入（QuantDinger已有模板）
- **尾盘抢筹策略**: 尾盘收盘在高位 + 放量 → 次日开盘卖出

## 工具文件

### data_loader.py
- **位置**: `optimizer/data_loader.py`
- **功能**: 数据加载、探查、校验、框架对接
- **用法**:
  ```powershell
  python data_loader.py explore    # 数据探查
  python data_loader.py validate   # 质量校验
  python data_loader.py load 000001 # 加载单只
  python data_loader.py list       # 列出全部代码
  ```

- **位置**: `optimizer/market_analysis.py`
- **功能**: 全市场统计分析（收益率分布、量价关系、板块轮动等）
- **用法**:
  ```powershell

  python market_analysis.py          # 完整模式（500只）
  ```
